Legend interactivity!

This is a working example of interactivity with the legends in animint. If you click any of the rows in either legend, the selected Species will update. This is a first implementation so feel free to provide suggestions.

Implementation

  1. First, I added a "legend" id to each of the legends. For example, the Sepal legend now renders as 
     < td id = "sepal_legend" >
        < table > 
           < tr id = "legend" > ... < /tr > 
        < /table > 
     < /td >
  2. Second, I added an id to each row. For example, the tr element for the versicolor row renders as 
     < tr id = "versicolor" > ... < /tr >
  3. Next, I created a function to select the id of the row in the legend when clicked by the user. Please note that this function is very temporary.
  4. Finally, I call this function at the end of add_legend().

Issues
